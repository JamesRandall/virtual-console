## Screen modes

Their are 4 available screen modes:

Mode 0: 256×160 @ 4bpp (widescreen, large)
Mode 1: 160×96 @ 8bpp (widescreen, colorful)
Mode 2: 128×128 @ 8bpp (square, compact)
Mode 3: 176×176 @ 4bpp (square, spacious)

Colours are selected from an indexed system into the full 256-color palette.

## Palette

Default 4bpp palette indexes:

```javascript
const paletteIndices = [
    242, // black
    255, // white
    6, // red
    61, // green
    127, // blue

    64, // dark green
    130, // dark blue
    52, // dark yellow
    229, // dark gray
    149, // purple
    60, // cyan
    170, // pink
    237, // stone
]
```
